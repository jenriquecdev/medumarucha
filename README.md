MEDUMARUCHA

A modern and responsive landing page developed for a food and beverage business in Guayaquil. The goal is to showcase products, highlight store locations, and streamline ordering via WhatsApp.

Demo

https://medumarucha.netlify.app/


This project was born from the need to digitalize a local business through a platform that is not only informative but also directly drives sales. The key pillars of development were:

    Strong visual identity: A custom themed interface inspired by Bikini Bottom to build brand recall.

    Conversion-focused: An optimized workflow for quick, friction-free ordering directly to WhatsApp.

    Performance and accessibility: A lightweight, fully responsive structure to ensure fast loading times on mobile devices.

Technologies Used

To keep the project agile, maintainable, and free of heavy build dependencies, a native and straightforward stack was chosen:

    Structure and styling: HTML5, CSS3, and Tailwind CSS (via CDN for fast, modular styling).

    Interactivity: Vanilla JavaScript (native JS) to handle logic and animations.

    Third-party components: Swiper.js for the interactive product carousel and Font Awesome for the site's iconography.

Main Features

    Themed UI: An immersive interface with smooth visual effects that elevate the user experience.

    Responsive Design: Complete adaptability to any screen size.

    Direct WhatsApp Ordering: A clean integration that connects the customer's purchase intent directly with the business's chat.

    Store Locations Section: An embedded map to help customers easily find physical stores.

    Smooth Navigation: A fixed navbar with internal scrolling to improve usability.

How to Run the Project

Since this is a project based on native client-side technologies, it does not require any dependency installations or complex runtime environments.

    Clone the repository:
    git clone https://github.com/tu-usuario/medumarucha.git

    Open the project:
    Simply open the index.html file in your preferred web browser.
